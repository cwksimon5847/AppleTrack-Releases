# AppleTrack Versions

## Version (Beta) 0.1 Info

 - **AppleTrack Protocol Version**: Pro

 - **AppleTrack Files**: toss.exe, data.exe
 - ***Toss.exe*** is the program that tracks the tosses
 - ***Data.exe*** is the program to convert BIN to CSV
